# React + Redux

Chat UI with React

## UI Concept

![Application concept](https://i.imgur.com/3I1aCx0.png)

### Prerequisites

To install all the dependencies
```
npm init
npm install
```

### Instructions

To start the watcher and keep update the compilation when code change

```
npm run watch
```

To development build

```
npm run dev-build
```

To build for code release

```
npm run build
```
